# Emacs.d

This is my emacs config forks from purcell&rsquo;s [config](http://github.com/purcell/emacs.d).


<a id="org97208ae"></a>

## Installation

To download this repository as your new emacs.d:

```sh
$ git clone https://github.com/iorest/emacs.d.git ~/.emacs.d
```

That&rsquo;s it!
